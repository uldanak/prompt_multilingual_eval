# Multilingual Prompt Engineering (English vs Kazakh)

This repository contains experiments for evaluating prompt engineering strategies in multilingual large language models.

## Tasks
- Sentiment Analysis (IMDB)
- Topic Classification (AG News)
- Commonsense Reasoning (CommonsenseQA)
- Question Answering (SQuAD)

## Strategies
- En-Basic
- Native-Basic
- Cross-lingual
- Translate-En
- Chain-of-Thought (for reasoning)

## Structure
- `data/`: datasets (English and Kazakh)
- `notebooks/`: experiment notebooks
- `results/`: evaluation outputs


## Model
GPT-4 (OpenAI API)
